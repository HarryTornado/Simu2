Trajectory in the State Space:

For 2nd order systems, this is simply the phase plane
portrait of the system.


For higher order systems, "traj" will project the state
space trajectory to various two dimensional subspaces.
For example, for a 3rd order system, there will be 3
phase plane portraits, from "x2-x1", "x3-x1", and "x3-x2"
subspaces.
