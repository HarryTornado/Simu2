On Creating Dynamic Systems

  1. Click the button on the tool bar or the corresponding item in the “Component” menu. A icon for the dynamic module will appear on the panel.

  2. You can drag the icon to appropriate position.

  3. You can double click the icon to bring forth the “DynamicEditor” as shown below

  1. Select the order of this dynamic module by pressing the “Number of State Equations” spinner to select the order of the module, and then press the corresponding “edit” button to edit the state equations. Then the following dialog window will appear

  1. You can also edit the output equation similarly.

  2. When entering the model for a linear system, you can first select the system order using the “Number of state equations” spinner and select the output number by the “number of output equations” spinner, then click the “tf” button to edit the transfer function.
